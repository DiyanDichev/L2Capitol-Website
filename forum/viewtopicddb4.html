Re: How to fix new patch?

by karnack » 29 Oct 2012, 9:51

there are few steps:

Go to control panel via startup menu.
Select network settings
Select local network (step 1. on pic)
Select properties (step 2. on pic)
Click on Internet protocol version 4 (TCP/IPv4) and select properties again (step 3 on pic.)
Select The manual selection of DNS server and write 8.8.8.8 (step 4. on pic)




After that, go to startup, search for "cmd" and turn on the commander. Write ipconfig/flushdns and press enter.
Restart pc and enjoy.
